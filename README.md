# Polychess
## Objectif :
Créer un jeux d'échec fonctionnement en console et graphique.

## Librairies requises : 
Ce jeux requière Python 3.7 ou au dessus.

Vous pouvez télécharger toutes les dépendance avec cette commande : 

`pip install -r Requirements.txt`
